== Installation Guide ==
{{TOC right}}

=== Enabling Shaders ===
1. Launch Raspberry Flavoured
2. Open any world
3. Navigate to: <code>Settings > Video Settings</code>
4. Click <code>Shader Packs...</code> (top-right)
5. Double-click your preferred shader to activate

=== Applying Custom Config ===
1. In <code>Shader Packs...</code> menu, click <code>Shader Pack Settings...</code>
2. Use the import button (top-right) to load the <code>.txt</code> config file

=== Troubleshooting ===
'''Problem''': Visual glitches (dark polygons)
* '''Solution''': Run command: <code>/flywheel backend instancing</code>
* If issues persist with hotbar items:
  1. Open any container (chest/inventory)
  2. If unresolved, restart the game

== Known Issues & Fixes ==
{| class="wikitable"
! Issue !! Solution
|-
| '''Sun/Moon missing''' || 
1. Disable <code>Fog Looks Good Now</code> mod<br>
2. ''or'' Navigate to <code>Settings > More Settings... > Mods & Configurations</code><br>
3. Search for the mod > <code>Config > Modify > Client</code><br>
4. Disable <code>Usecavefog</code>
|-
| '''Recipe book items appear darkened''' || 
1. Open inventory > Click gear icon (bottom-left)<br>
2. Search for <code>Use Batch Renderer</code><br>
3. Set to <code>False</code>
|}
